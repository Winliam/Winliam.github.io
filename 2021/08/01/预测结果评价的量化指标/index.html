预测结果评价的量化指标

Positive/Negative

样本空间的原始分类;

True/False

预测结果与样本原始属性的偏差情况，即P->P/N->N为T，P->N/N->P为F;因此，T和F需要再细分为：

  • TP：P->P(正类被预测为正类，真正类)
  • TN：N->N(负类被预测为负类，真负类)
  • FP：N->P(负类被预测为正类，假正类)
  • FN：P->N(正类被预测为负类，假负类）

变量命名(personal)

若以P为样本中正类数量，N为负类数量，M为样本总数，p为预测结果中的正类数量，n为负类数量，T为正确预测数量，F为错误预测数量，综上有：

  • M = P+N = p+n = T+F（不解释）
  • T = TP+TN（不解释）
  • F = FP+FN（不解释）
  • P = TP+FN（样本中的正类数量=真正类数量+假负类数量）
  • N = TN+FP（样本中的负类数量=真负类数量+假正类数量）
  • p = TP+FP（预测中的正类数量=真正类数量+假正类数量）
  • n = TN+FN（预测中的负类数量=真负类数量+假负类数量）

基本指标

  • accuracy
    T/M，模型做正确预测的能力
  • recall/hit rate/TruePositiveRate/sensitivity
    TP/P，原始样本中的正类有多少被正确预测出来
  • specificity/TrueNegativeRate
    TN/N，原始样本中的负类有多少被正确预测出来
  • fall-out/FalsePositiveRate
    FP/N，原始样本中的负类有多少被错误的预测成了正类
  • precision
    TP/p，找到的正类中有多少是真正的正类
  • F1 score
    precision和recall的调和平均数

ROC曲线

  • 横轴为FPR，样本中负类多少被错误预测成了正类
  • 纵轴为TPR，样本中正类多少被正确预测成了正类（也就是recalll）
  • 0-1之间选择不同的分类阈值，从而得到同一分类器的不同subversion，因此ROC曲线上一个点(x,y)的含义为：对于某个特定的分类阈值，有y%的正类被成功预测成了正类，但代价是x%的负类也被预测成了正类。
  • 我们肯定希望x越小越好，y越大越好（所有的飞机都被找到，大雁还不被当成飞机）
  • AUC值，ROC曲线对x轴的积分，对于同样的x值，y值越大，面积越大，越符合我们的期望，也就是分类效果越好。

P-R曲线

  • 横轴为TPR/recall（真正类占全体样本正类的比例）
  • 纵轴为precision（真正类占全体预测正类的比例）
  • 0-1之间选择不同的分类阈值，从而得到同一分类器的不同subversion，因此P-R曲线上一个点(x,y)的含义为：对于某个特定的分类阈值，有x%的正类被成功预测成了正类，并且这些真正类占正预测的比例为y%。
  • 我们希望x和y都越大越好，因为这意味着所有的飞机都被找到的同时，大雁还不被当成飞机（这特么的跟ROC曲线含义一样啊）

ROC曲线和PR曲线对比

  1. 选定一个分类阈值，那么此时的recall是相同的，那么对于达到这个recall效果:
    ROC曲线中衡量的是FPR（FP/N），也就是样本中的这么多大雁有多少被错当成了飞机
    PR曲线中衡量的是precision，也就是预测中的这么多飞机有多少真的是飞机（或者说有多少其实是大雁）
  2. 如果负样本增加：
    • 预测结果中，由于分类模型没有变，所以真负类数量和假正类数量必然会同时增加，因此FPR总体变化不会很大
    • 预测结果中，真正类数量不会变化，但是假正类数量必然增加，precision总体会变小
  3. 如果正样本增加：
    • 真负类数量不会发生变化，假正类数量也不会发生变化，因此FPR不变
    • 假正类数量不变，真正类数量必然增加，precision总体会变大
    • 综上所述，我们说，样本的分布情况会影响PR曲线形状，但（基本）不会影响ROC曲线形状；也因此，正负样本分布极不均匀时，优先选择PR曲线评估分类模型的性能（ROC曲线特喵的不受样本分布影响啊）
